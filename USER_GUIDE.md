## ğŸ”§ Troubleshooting

### Can't Join Room
- **Check room code** (6 characters)
- **Verify internet connection**
- **Ask host** if room is still active
- **Try again** - tap Join Room

### Scores Not Updating
- **Check internet** connection
- **Wait a moment** - sync may be delayed
- **Refresh app** if needed

### Timer Not Starting
- **Host must start** - Use timer controls
- **Check Host Controls** panel
- **Tap "Start"** button

### Physical Challenge Stuck
- **Host**: Use "Approve Challenge" in controls
- **Players**: Wait for host approval
- **Solution**: Host can skip to next question

### Match History Empty
- **Complete at least one game** first
- **Wait 2-3 seconds** for sync
- **Check internet** connection
- **Firestore index** may still be building (wait 2-5 min)
